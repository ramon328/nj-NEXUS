# Clientes — Guía de Usuario

## Acceso
Menú lateral → **Clientes**

## ¿Qué puedes hacer?

### Ver clientes
La tabla muestra todos tus clientes con nombre, RUT, email, teléfono y fecha de registro. Usa la barra de búsqueda para encontrar rápidamente.

### Agregar un cliente
Click en **"+ Nuevo cliente"**. Primero elige el **tipo de cliente**:
- **Persona**: completa nombre, apellido, RUT, email, teléfono y fecha de nacimiento.
- **Empresa**: completa **razón social** y **RUT** de la empresa (en vez de nombre y apellido).

El tipo de cliente se puede elegir desde cualquier lugar donde agregues un cliente (al comprar un auto, vender, reservar, cotizar o desde la página de Clientes).

### Editar un cliente
Click en un cliente de la tabla para editar su información.

### Importar desde Excel
Si ya tienes una base de clientes, puedes importarla masivamente:
1. Click en **"Importar"**
2. Sube tu archivo Excel
3. Mapea las columnas
4. Confirma la importación

## ¿Dónde se usan los clientes?

Los clientes se seleccionan al:
- Registrar una **venta**
- Crear un **financiamiento**
- Registrar un **lead**
- Hacer una **reserva**
- Generar **documentos** (cotización, compraventa, etc.)
